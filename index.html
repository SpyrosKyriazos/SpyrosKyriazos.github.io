Projects

Car vs Bus

Car vs Bus is a web app that calculates the cost of fuel for your everyday commute and compares it to the cost of public transport.

Available here (In Greek).

E-Prescribing

E-Prescribing is an electronic prescription system writen in Java using the Vaadin application framework and JPA.

Car Strikers

Car Strikers is a simple football/soccer game but the human players are replaced with cars. It is a multiplayer only game and every player controls one car. It was part of my final year project in university.

Available here for download (zip file containing the game jar).

Geolocation related content

Project created during my Master's Thesis. It contains an editor to create a floor plan and assign it to a geographical location and an Android client where the user can navigate the floor in 3D space.

Document available here (In Greek).

Mobile Gamer Simulator

Mobile Gamer Simulator is a game where the player controls the finger of a mobile gamer, using the arrows to move and spacebar to tap. Created during Ludum Dare 31.

Play it here.

Escape From Beneath

Escape From Beneath is a platformer where you try to escape a cave system after a cave-in. Created during Ludum Dare 29.

Play it here.

Hungry Dragon

In Hungry Dragon a dragon is always following the player and the player must lead the dragon to his food in less than 10 seconds. Created during Ludum Dare 27.
